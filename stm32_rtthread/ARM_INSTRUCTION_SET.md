# ARM Cortex-M (Thumb-2) 指令全集速查 (按常用频率分类)

此表专为具有 x86 基础的嵌入式开发者设计，涵盖了从 RT-Thread 内核开发到外设驱动编写最常遇到的指令。

---

## 频率 1：核心高频 (几乎每一行代码背后都有它们)
这些指令构成了程序的骨架：数据搬运、基本计算和流程控制。

| 指令 | 描述 | x86 参考 | 嵌入式应用场景 |
| :--- | :--- | :--- | :--- |
| **LDR** | **Load**: 从内存读到寄存器 | `MOV EAX, [mem]` | 获取全局变量、访问外设寄存器地址 |
| **STR** | **Store**: 从寄存器写回内存 | `MOV [mem], EAX` | 修改变量、配置外设寄存器 |
| **MOV** | **Move**: 寄存器间传值 / 立即数 | `MOV` | 函数参数传递、局部变量初始化 |
| **ADD/SUB**| 加法 / 减法 | `ADD / SUB` | 指令指针偏移、计数器累加 |
| **B / BL** | 跳转 / 带返回跳转 (Calling) | `JMP / CALL` | 函数调用、死循环、GOTO |
| **BX** | 带状态切换跳转 (Branch Exchange)| `RET` | 常用于 `BX LR` 实现函数返回 |
| **PUSH/POP**| 压栈 / 出栈 | `PUSH / POP` | 函数入口/出口保存寄存器 R4-R11 |

---

## 频率 2：逻辑与位操作 (驱动编写的核心)
在操作 GPIO、I2C、SPI 或配置寄存器位域时必不可少。

| 指令 | 描述 | x86 参考 | 嵌入式应用场景 |
| :--- | :--- | :--- | :--- |
| **BIC** | **Bit Clear**: 按位清除 | `AND EAX, ~imm` | 清除特定的寄存器控制位 |
| **ORR/AND** | 按位或 / 按位与 | `OR / AND` | 设置或检测特定的寄存器位 |
| **LSL/LSR** | 逻辑左移 / 逻辑右移 | `SHL / SHR` | 拼接数据包（如传感器读取结果） |
| **ASR** | 算术右移 | `SAR` | 带符号整数除以 2 的幂次 |
| **CMP** | 比较 (更新标志位) | `CMP` | `if/while` 条件判断的前置操作 |
| **CBZ/CBNZ**| 比较为零/非零则跳转 | `CMP+JZ/JNZ` | 指针非空检查、循环计数值检查 |
| **TST** | 测试位 (按位与并更新标志) | `TEST` | 检查状态字中的某一位是否置 1 |

---

## 频率 3：系统与 RTOS 关键指令 (深入内核必看)
涉及到中断控制、线程切换和存储隔离。

| 指令 | 描述 | x86 参考 | 嵌入式应用场景 |
| :--- | :--- | :--- | :--- |
| **MRS** | 读特殊寄存器 (Move from SR) | `(无)` | 读取栈指针 `PSP` 或 `MSP`，读中断屏蔽器 |
| **MSR** | 写特殊寄存器 (Move to SR) | `(无)` | 切换线程栈、恢复中断状态 |
| **CPSID I** | 关全局中断 (Disable) | `CLI` | 临界区保护 (RT-Thread 的 `rt_hw_interrupt_disable`) |
| **CPSIE I** | 开全局中断 (Enable) | `STI` | 退出临界区 |
| **ISB** | 指令同步隔离 (Flush Pipeline)| `(无)` | 更改系统配置（如 MPU/系统频率）后强制同步 |
| **DSB** | 数据同步隔离 (Memory Barrier)| `MFENCE` | 确保 DMA、外设写操作完全完成后再继续 |
| **LDM/STM** | 多寄存器加载/存储 | `(多次PUSH/POP)` | RT-Thread 线程上下文切换时保存所有寄存器 |

---

## 频率 4：高级/优化指令 (追求性能与精简)
在高性能库、信号处理或底层驱动优化中使用。

| 指令 | 描述 | 类似功能 | 嵌入式应用场景 |
| :--- | :--- | :--- | :--- |
| **REV** | 字节序反转 (Byte Reverse) | `BSWAP` | 大端/小端转换（如网络包解析） |
| **BFI/UBFX**| 位域插入 / 位域提取 | `Shift + Mask` | 极其优雅地操作寄存器中间的 3-5 位 |
| **MUL/MLA** | 乘法 / 乘累加 | `IMUL` | PID 控制算法、滤波器计算 |
| **UDIV/SDIV**| 除法 | `DIV` | 计算频率、比例缩放 (注意 F1 系列不支持) |
| **CLZ** | 计算前导零个数 | `BSR` | 实时系统中确定优先级 (找到最高优先级任务) |
| **IT** | If-Then (条件执行块) | `(无)` | 将简单的 `if-else` 转为高效的无分支代码 |

---

### ARM (Cortex-M) 特有的寄存器速记：
*   **R13 (SP)**: 栈指针。Cortex-M 有两个：MSP (主栈) 和 PSP (进程栈)。
*   **R14 (LR)**: 链接寄存器。存函数返回地址。x86 是存栈里，ARM 指令更快。
*   **R15 (PC)**: 程序计数器。修改它就是 `JMP`。

---
**RT-Thread 提示**：在 `context_gcc.S` 中，`PSP` 指向的是线程栈，而 `MSP` 指向的是系统栈（中断使用）。
